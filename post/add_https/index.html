苦等24小时无果秒转阵地 #Netlify 不用繁琐等待只要10分钟左右就安全加锁
1.注册一个 Netlify 帐号，地址 Netlify App 选择用 GitHub 注册就好了。 2.新建一个站点 3. 然后选用github根据提示 1 2 3步就好 4.由于是把你的项目从git上面克隆下来,但是名字是随机生成的所以需要你自己修改一个短名 5.修改好了,但是这个https是它提供的和git一样的接下来要迁移到我们自己的域名

6.在这里输入你自己的域名,但是肯定会报错因为并没有解析,会给你一个DNS地址你需要添加到你自己的云解析里面 7.添加到云解析后再返回来保存然后选择下面的https点击第一个根据提示会出现小圆锁转动，等待ing 8.会提示https成功 9.最后,看一看原谅锁
